CPU (Central Processing Unit)

The CPU is the brain of the computer. It performs calculations and executes instructions.

Monitor

The monitor displays visual output like text, images, and videos.

Keyboard

The keyboard is used to input data by typing.

Mouse

The mouse helps you point, click, and interact with the screen.

RAM (Memory)

RAM stores temporary data and improves speed.

Hard Disk

The hard disk stores data permanently.
